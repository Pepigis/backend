Úvod do PHP
PHP (zkratka pro "PHP: Hypertext Preprocessor") je populární open-source skriptovací jazyk, který běží na straně serveru. Je zdarma, výkonný a pohání některé z největších blogovacích systémů a sociálních sítí na webu.

Proč používat PHP?
Multiplatformní: Běží na všech hlavních operačních systémech (Windows, Linux, Unix, Mac OS X atd.).

Kompatibilita: Funguje se všemi předními webovými servery (Apache, Nginx, IIS atd.).

Podpora databází: Podporuje širokou škálu databází (MySQL, PostgreSQL, MongoDB, Oracle a další).

Zdarma a efektivní: Je zdarma ke stažení na www.php.net a efektivně běží na serveru.

Snadné učení: Ideální jako první server-side jazyk pro začátečníky.

Co byste měli umět předem
Před zahájením práce s PHP je vhodné mít základní znalosti:

HTML

CSS

JavaScript

Co je to PHP soubor?
Má příponu .php.

Může obsahovat text, HTML, CSS, JavaScript a PHP kód.

Kód se spouští na serveru a výsledek se vrací do prohlížeče jako čisté HTML.

Co PHP dokáže?
Generovat dynamický obsah stránek.

Vytvářet, číst, zapisovat, mazat a zavírat soubory na serveru.

Sbírat data z formulářů.

Posílat a přijímat cookies.

Přidávat, mazat a upravovat data v databázi.

Řídit uživatelské přístupy a šifrovat data.

Generovat různé výstupy (nejen HTML, ale i obrázky, PDF, XML atd.).

Novinky v PHP 8
Verze PHP 8 přináší významná vylepšení výkonu a syntaxe:

JIT (Just-In-Time) Compiler: Pro rychlejší provádění kódu a lepší výkon.

Union Types: Umožňují proměnným a návratovým hodnotám akceptovat více datových typů.

Named Arguments (Pojmenované argumenty): Umožňují předávat argumenty funkcím podle názvu, ne jen podle pořadí.

Match Expressions: Výkonnější a stručnější alternativa k příkazu switch.

Nullsafe Operator (?->): Snadný přístup k vlastnostem objektu, který může být null.

Attributes (Atributy): Možnost přidávat metadata přímo do kódu.

Nové funkce: Například str_contains(), str_starts_with() a další.
